# Visibility

## show
```javascript
vff.show(template)
```
Set “visibility” property in the template to true

* template - _string_ - the name of the template

## hide
```javascript
vff.hide(template)
```
Set “visibility” property in the template to false

* template - _string_ - the name of the template

## toggle
```javascript
vff.toggle(template)
```
Toggle “visibility” property in the template

* template - _string_ - the name of the template
